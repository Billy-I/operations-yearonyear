# Operations Year-on-Year Prototype

A React and TypeScript prototype created to explore how users could compare farm operations and cost data across multiple years.

The project was developed as an interactive UX prototype for usability testing and product validation.

## Purpose

The prototype explored how agricultural users could:

- compare operational performance across different years
- identify changes in costs and activity
- filter complex datasets
- switch between summary and detailed views
- understand trends through charts and tables

## Features

- Year-on-year comparisons
- Multi-year analytics
- Cost metrics and summaries
- Interactive charts
- Filters and data controls
- Responsive dashboard layouts
- Reusable interface components

## Technology

- React
- TypeScript
- Vite
- Tailwind CSS
- Vercel
